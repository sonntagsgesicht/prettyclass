Python Project *prettyclass*
-----------------------------------------------------------------------


Introduction
------------

To import the project simply type

.. code-block:: python

    >>> import prettyclass

after installation.


Install
-------

The latest stable version can always be installed or updated via pip:

.. code-block:: bash

    $ pip install prettyclass
